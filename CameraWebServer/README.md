
Upload the code in arduino ide 
Before uploading click the reset buttons and close serial monitor if it is open.

After the upload is complete, click the reset buttons again. 
You will get an IP address. Enter the ip address in the web browser.

After the web page is loaded, scroll down and click "Start Streaming".


If you encounter any problem: https://randomnerdtutorials.com/esp32-cam-troubleshooting-guide/
